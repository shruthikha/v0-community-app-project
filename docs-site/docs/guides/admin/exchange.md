# Exchange Moderation

As an administrator, you have the tools to manage the community exchange, moderate flagged content, and ensure a safe trading environment.

## 📊 The Admin Dashboard
The Admin Exchange table provides a global view of all listings in the community.

![Admin Table](/screenshots/exchange_admin_listings_table_5.png)

### Key Features:
- **Status Filtering**: View listings by "Published", "Draft", or "Archived" status.
- **Creator Context**: See which resident created each listing.
- **Flag Counts**: Items flagged by residents are prioritized in the moderation queue.

---

## 🚩 Handling Flags
Residents can flag items for various reasons (prohibited content, spam, etc.). 

### Moderation Workflow:
1.  **Review**: Select a flagged listing to view details and flag reasons.
2.  **Clear Flags**: If the listing is compliant, use the "Clear Flags" action to reset the counter.
3.  **Archive**: If a listing is inappropriate, archive it. This hides it from the community but preserves the record for audit.

![Bulk Actions](/screenshots/exchange_admin_bulk_moderation_6.png)
*Administrators can perform bulk actions like Archiving or Clearing Flags on multiple listings at once.*

---

## 🗑️ Archiving vs Deleting
- **Archiving**: This is the preferred method for moderation. It hides the item and cancels any pending (unfilled) transactions, but keeps the data available for future reference.
- **Deleting**: Only possible if there are no active pickup/return transactions. Deleting is permanent and cannot be undone.
