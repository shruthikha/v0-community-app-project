# Events Management

As a community administrator, you can oversee all resident-created events, manage official community calendars, and moderate content to ensure a safe and vibrant environment.

---

## 🛡️ The Events Dashboard

Access the master events list at **Admin > Events**.

![Admin Events Table](/screenshots/admin_events_table_01.png)

### Moderation & Flagging
Residents can flag events they believe are inappropriate or inaccurate.
- **Flag Count**: The dashboard highlights events with active flags using a red warning icon.
- **Review**: Admins can review flagged events and choose to **Dismiss** the flag or **Delete** the event if it violates community standards.

### Bulk Actions
Admins can select multiple events to perform bulk deletions or status updates, improving efficiency for large community cancellations.

---

## 🏷️ Category Management

Admins have full control over the taxonomy used to tag and filter events.

### Managing Categories
The category list shows all available event types along with their icons and visibility settings.

![Category List](/screenshots/admin_events_categories_02(1).png)

### Creating a Category
Creating a new category allows you to define the visual representation and default behavior for a specific type of event.

![Category Creation Form](/screenshots/admin_events_categories_02(2).png)

- **Label**: The display name for the category (e.g., 'Wellness', 'Town Hall').
- **Icon**: Choose a standard Lucide icon or emoji to represent the category.
- **Color**: Set a primary color for the category marker on the map.
- **Status**: Toggle between `active` or `deprecated` status.

---

## 🗓️ Official Community Events

Admins can create **Official Community Events** which appear with priority in the resident feed and cannot be flagged.

- **Creation**: Use the standard event form but select the **Official** type.
- **Recurrence**: Official events often use robust series logic (e.g., "Monthly HOA Meeting").

> [!IMPORTANT]
> Official events are inherently "Trusted" and circumvent the flagging system. They are displayed with a special verification badge to residents.
