# Monitoring & Follow-up

Once submitted, you can track the progress of your request in real-time.

## Status Updates

You will receive notifications whenever an admin updates the status of your request (e.g., moving it to "In Progress" or "Resolved").

### Common Statuses
*   **Pending**: Received and waiting for initial review.
*   **In Progress**: Being actively addressed by the administration.
*   **Resolved**: Successfully completed.
*   **Rejected**: Cannot be fulfilled (a reason will be provided).

## Conversation Threads

Each request has a dedicated comment section. You can use this to:
*   Reply to admin questions.
*   Provide more information or photos.
*   On public requests, see comments from other neighbors who might be experiencing the same issue.
