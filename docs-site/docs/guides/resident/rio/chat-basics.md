# Chatting with Río

Río is your friendly neighborhood AI assistant, designed to help you navigate community life and find information quickly.

## How to Start a Chat

![Río Welcome Card](/screenshots/rio_welcome_card_step_1.png)
_Introducing Río: Your friendly neighborhood AI._

You can find Río in the main navigation or via the **Create/Action Menu** at any time.

![Río Action Menu](/screenshots/rio_create_menu_step_1.png)
_Easy access via the create menu._

## Key Chat Features

### 1. Smart Memory
Río remembers what you've discussed earlier in the conversation, so you don't have to repeat yourself. If you refresh the page or come back later, Río will pick up right where you left off.

### 2. Citations & Transparency
When Río uses information from [official community documents](../documents.md) (like bylaws or rules), it will include a small numbered citation [1]. Click it to see the source document and an excerpt of the relevant section.

![Río Citations UI](/screenshots/rio_citations_ui_step_1.png)
_Citations ensure you're getting official information._

### 3. Cheat Sheet
Not sure what to ask? Use the **Cheat Sheet** for quick suggestions on common neighbor topics.
