# Neighbor Lists & Group Interactions

Neighbor Lists are private, personal groups you create to organize your connections and simplify community life.

## Managing Your Personal Lists

You can create lists to group neighbors for easy reference (e.g., "Dog Walkers" or "Second Floor Neighbors").

![Create List Modal](/screenshots/resident_create_list_step_1.png)

1.  **Create**: Click "Create New List" and give it a descriptive name.
2.  **Add Members**: Search and select the neighbors you want to include.
3.  **Edit**: Use the "Edit List" option to modify membership as your connections grow.
    ![Edit List Modal](/screenshots/resident_edit_list_step_1.png)

## Using Lists for Invitations

Neighbor Lists serve as "Smart Selectors" throughout the platform:

- **Events**: When organizing a community event, you can toggle an entire Neighbor List to invite everyone at once.
- **Check-ins**: Use lists to quickly select a group for security or wellness check-ins.
- **Social Proof**: The Events dashboard shows if people from your private lists are attending, helping you find events where you'll already know a friend.

> [!NOTE]
> All Neighbor Lists are **private to you**. Other residents cannot see your lists or who you've grouped together.
