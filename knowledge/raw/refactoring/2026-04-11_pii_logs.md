# Redact user IDs in AI chat logs

## Finding
`app/api/v1/ai/chat/route.ts` logs user IDs directly in console.error messages. Wiki pattern mandates PII redaction in production logs.

## Files
- `app/api/v1/ai/chat/route.ts`

## Recommendation
Replace direct user ID logging with hash or redact:
```typescript
// Bad
console.error(`User ${user.id} attempted...`)
// Good
if (process.env.NODE_ENV !== 'production') {
  console.log('[debug] User:', user.id)
}
```
